🎵 Spotify Clone

A feature-rich music player inspired by Spotify, built using HTML, CSS, and vanilla JavaScript.

🔗 Live Demo

👉 https://vasanth038.github.io/Spotify-Clone .

## 🚀 Features
- Dynamic rendering of 18 songs
- Play / Pause, Next / Previous controls
- Shuffle and Repeat modes
- Real-time progress bar with timer sync
- Volume control with mute toggle
- Keyboard shortcuts for seamless control

  ## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
- Font Awesome
